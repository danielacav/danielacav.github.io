HTML tags learnt

Emphasis
A string of emphasized, slightly more important text. Screen readers will change their voice for this text.
Highlighting text
Used to highlight a piece of text for reference. The keywords in a search results page, the current navigation item, etc
Superscript text
Defines a text as Superscript: I was born the 21st of July
Address
Useful when adding contact information, email, tel, postal address, etc.
1345 Woodroffe Avenue
Keyword
Defines a keyword, like product name in a review, a lead sentence in a paragraph.
